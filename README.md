Blink Runner is a retro-style arcade runner where your character moves automatically and jumps based on sound input from your microphone. Players control the jumps and double jumps by generating noise (clapping, shouting, or using musical instruments). The game combines a neon 8-bit aesthetic with a fun and dynamic sound-reactive mechanic.

How to Play:

Open the webpage.

Click Start to begin the game.

Make a sound above the green threshold to perform the first jump.

Make a louder sound above the red threshold to perform a double jump.

Avoid obstacles and try to get the highest score possible.

Click Stop to pause or Restart after dying.

Development Notes:

Built using HTML5 Canvas and JavaScript.

Sound input handled via navigator.mediaDevices.getUserMedia and Web Audio API.

Obstacles and player rendering use simple shapes for retro aesthetic.

Designed to be lightweight and run smoothly in modern browsers.
